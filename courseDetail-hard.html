困難主題

  使用函數方法和遞迴能畫出複雜幾何圖形(The Recursive Tree)
  The Recursive Tree需透過下列步驟:
- 新增函數
function draw(startX, startY, len, angle) {
  ctx.beginPath();
  ctx.save();
  ctx.translate(startX, startY);
  ctx.rotate(angle * Math.PI/180);
  ctx.moveTo(0, 0);
  ctx.lineTo(0, -len);
  ctx.stroke();
  if(len < 40) {
    ctx.restore();
    return;
  }
  draw(0, -len, len*0.8, -15);
  draw(0, -len, len*0.8, +15);
  ctx.restore();
}
- 繪製單一樹幹、分支
  使用 translate()、 rotate()、 moveTo() 方法來移動畫布、原點來控制樹的生長方向。
  ctx.translate(startX, startY);
  ctx.rotate(angle * Math.PI/180);
  ctx.moveTo(0, 0);
  ctx.lineTo(0, -len);
  ctx.stroke();
- 增加遞迴條件式
  透過遞迴方法增生出茂密的樹
  if(len < 40) {
    ctx.restore();
    return;
  }
