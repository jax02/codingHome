關於 Canvas

Canvas 是 HTML5 的新元素，可以形容為一個畫布容器，本身無繪製能力，可透過 JavaScript 繪製圖形。
包含繪圖、文字、合成圖照片、建立動畫、甚至處理即時的影片播放、遊戲及動畫製作。

資料視覺化

Canvas可以用於創建各種資料視覺化圖表和圖形，例如折線圖、柱狀圖、餅圖等等，可以讓數據更加生動直觀地呈現。

遊戲開發

Canvas可以用於開發網頁遊戲，因為它可以創建動畫效果，例如遊戲角色的移動、特效等等。

繪圖工具

Canvas可以用於創建繪圖工具，例如繪製線條、曲線、圖形等等。

地圖應用

Canvas可以用於創建交互式地圖應用，例如地圖標記、定位等等。

廣告應用

Canvas可以用於創建動態網頁廣告，例如製作動畫廣告等等。

音樂視覺化

Canvas可以用於創建音樂視覺化應用，例如創建動態音樂視覺效果等等。
